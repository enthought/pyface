pyface.i_layout_widget module¶

class pyface.i_layout_widget.ILayoutWidget(adaptee, default=<class 'traits.adaptation.adaptation_error.AdaptationError'>)[source]¶

Bases: pyface.i_widget.IWidget, pyface.i_layout_item.ILayoutItem

Interface for widgets that can participate in layout.

Most widgets implement ILayoutWidget, but widgets like top-level windows, menus, toolbars, etc. do not.
